# R Package Requirements

This project requires the following R packages:

## Core data manipulation
- dplyr
- tidyr
- purrr
- tibble

## File I/O
- readr

## Date handling
- lubridate

## String handling
- stringr

## Visualization
- ggplot2

## Installation

Install all required packages with:

install.packages(c(
  "dplyr",
  "tidyr",
  "purrr",
  "tibble",
  "readr",
  "lubridate",
  "stringr",
  "ggplot2"
))
