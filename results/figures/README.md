# Figures

This folder contains final visualization outputs generated by the analysis pipeline.

## Note on appearance

Some figures may appear differently depending on the viewer:

- GitHub (especially in dark mode) may display figures with a dark background or stronger gridlines
- This is due to transparent backgrounds in PNG files

All figures are generated with a light (white) background and are best viewed:
- locally
- or when downloaded

## Contents

- `iafd/` → intra-annual flow distribution plots
- `fdc/` → flow duration curve plots
- other folders → additional indicator visualizations

All figures can be reproduced by running the scripts in the `scripts/` directory.
